# AdventureWorks Sales Analytics

## Project Overview
This is an End-to-End Data Analyst Project built using SQL, Python, and Power BI.

The project focuses on analyzing AdventureWorks sales data to identify revenue trends, customer behavior, product performance, and business insights.

## Tools Used
- SQL Server
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Project Workflow

### 1. SQL
- Data extraction
- Joins
- Aggregations
- Customer analysis
- Product analysis

### 2. Python
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Missing value treatment
- Outlier detection using IQR
- Feature Engineering

### 3. Power BI
- KPI Dashboard
- Revenue Analysis
- Customer Analysis
- Product Performance Analysis

## Key KPIs
- Total Revenue
- Total Orders
- Total Customers
- Average Order Value

## Key Insights
- A small number of products generated most of the revenue.
- High-value customers contributed significantly to total sales.
- Revenue increased with larger order values.
- Some territories performed better than others.
